# EmbyConnect

Connects your, nearly useless, ATV 3 to Emby Media Server.

## Getting Started

Install dependencies with:

    $ npm install
    
after this run EmbyConnect with:

    $ sudo ./EmbyConnect.js
    
## Commandline Options

* **--host**

  The hostname to hijack
  
* **-e, --emby**

  The Emby Media Server address
  
* **--skipSSL**

  Starts server without SSL.
